1. Objects - pwede tayong maglagay ng iba't ibang data dito.

2. Built in objects:
	JSON - JSON can store data and use it universally.
	localStorage - we can store data permanently here.

3. What we can do in JSON?
	JSON.stringify(objects) - converts object into string
	JSON.parse(data) - converts string into data

	Example for JSON.stringify(objects)

		const people = {
			name: 'George',
			age: 25
		}

		JSON.stringify(people);

		output: '{"name":"George","age":25}'
        
	Example for JSON.parse(data)

		const people = {
			name: 'George',
			age: 25
		}

		JSON.stringify(people);
		JSON.parse(people);

		output: object = {name: 'George', age: 25}
